# 🧠 Image Generation from Text using GAN

## 📌 Project Overview

This project implements **Text-to-Image Generation** using a **Generative Adversarial Network (GAN)**.

The model generates images of fashion items based on input text labels.

---

## 🎯 Objective

To train a GAN model that learns the relationship between text labels and image features and generates realistic images.

---

## 🛠️ Technologies Used

* Python
* TensorFlow / Keras
* NumPy
* Matplotlib

---

## 🧠 Model Architecture

* Generator: Creates images from noise + label
* Discriminator: Classifies real vs fake images
* GAN: Combined model for training

---

## 📂 Dataset

* Fashion MNIST Dataset
* 10 classes (T-shirt, Shoe, Bag, etc.)

---

## ▶️ How to Run

1. Open the notebook in Google Colab
2. Run all cells
3. Train the GAN model
4. Generate images
